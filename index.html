Installazione alpaka

Clone the repository

git clone https://github.com/alpaka-group/alpaka.git

Installing boost (>1.74)

TMP e' la cartella che e' stata creata dal comando mktemp -d

mktemp -d 
curl -L -s -S https://boostorg.jfrog.io/artifactory/main/release/1.78.0/source/boost_1_78_0.tar.bz2 | tar xj -C TMP
cd TMP/boost_1_78_0
./bootstrap.sh
./b2 install --prefix=INSTALL_DIR --without-graph_parallel --without-mpi --without-python
rm -rf TMP

Installing gcc (> 9.5)

In qyesto caso prendiamo gcc11 da cvmfs: source /cvmfs/sft.cern.ch/lcg/contrib/gcc/11.2.0/x86_64-centos7/setup.sh

Compile examples

Su cpu

g++ -O2 -std=c++17 \
-DALPAKA_ACC_CPU_B_SEQ_T_SEQ_PRESENT  -DALPAKA_ACC_CPU_B_SEQ_T_SEQ_ENABLED  -DALPAKA_ACC_CPU_B_SEQ_T_SEQ_SYNC_BACKEND \
-isystem alpaka/include -isystem boost/include \
alpaka/example/monteCarloIntegration/src/monteCarloIntegration.cpp -o monteCarlo

Su gpu invidia (cuda > 11.0)

/usr/local/cuda-11.4/bin/nvcc -O2 -x cu \
-DALPAKA_ACC_GPU_CUDA_PRESENT -DALPAKA_ACC_GPU_CUDA_ONLY_MODE -DALPAKA_ACC_GPU_CUDA_ENABLED -DALPAKA_ACC_GPU_CUDA_ASYNC_BACKEND -UALPAKA_HOST_ONLY \
-isystem alpaka/include -isystem boost/include \
alpaka/example/monteCarloIntegration/src/monteCarloIntegration.cpp -o monteCarlo
